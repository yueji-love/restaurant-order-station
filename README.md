# 餐厅点单台

面向餐厅前台与出餐员工的响应式点单工作台。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 已实现

- 号码、品类、小料、确认四步点单流程
- 使用中、已选择、未选择等可辨识状态
- 下单成功反馈与自动重置
- 出餐队列的制作、通知取餐、完成取餐状态
- 出餐排序与提示音设置界面
- 桌面、iPad 和手机响应式布局
- 键盘焦点、语义化控件和减少动态效果支持

视觉意向图位于 `design-reference/order-entry-concept.png`，完整设计令牌和交互规则位于 `DESIGN.md`。
