# 餐厅点单台

面向餐厅前台与出餐员工的响应式点单工作台。

## 本地运行

```bash
npm install
npm run dev
```

开发模式会同时启动网页和实时接口。局域网内的其他设备访问开发电脑显示的局域网地址，即可实时共享订单和设置。

生产构建：

```bash
npm run build
npm start
```

生产模式默认监听 `5175` 端口。其他设备可访问 `http://开发电脑IP:5175`。

## 已实现

- 号码、品类、小料、确认四步点单流程
- 使用中、已选择、未选择等可辨识状态
- 下单成功反馈与自动重置
- 多设备订单实时同步
- 出餐队列的开始制作和完成取餐操作
- 可持久化的出餐排序与提示音设置
- 桌面、iPad 和手机响应式布局
- 键盘焦点、语义化控件和减少动态效果支持
